# AI Coding Challenge 2026 - Mirror Project

Sistema di orchestrazione swarm e self-healing basato su LangChain e OpenAI.

## 🚀 Utilizzo

1. **Attiva l'ambiente virtuale**:
   ```bash
   source .venv/bin/activate
   ```
2. **Installa le dipendenze**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Esegui l'orchestratore**:
   ```bash
   python -m execution.swarm_orchestrator
   ```

## 📁 Struttura
- `execution/`: Script core per self-healing e orchestrazione (con `__init__.py`).
- `Dataset/`: Risorse e dati per il challenge.
- `Guides/`: Documentazione tecnica e glossari.
